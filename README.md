# BloxPowers

Claude Code plugin за автоматизирано участие в обществени поръчки (ЕОП).

Автоматизира целия процес — от сканиране на портала за нови поръчки, през анализ на документацията и ценообразуване, до генериране и преглед на финалната оферта.

---

## Инсталация

```bash
/plugin marketplace add blox-eng/bloxpowers
/plugin install bloxpowers@bloxpowers
```

---

## Предварителни изисквания

| Зависимост | Описание |
|---|---|
| Playwright MCP plugin | Браузър автоматизация за достъп до ЕОП портала |
| python-docx | Четене и генериране на Word документи |
| openpyxl | Четене и запис на Excel файлове |
| LibreOffice | Конвертиране на документи |

---

## Налични skills

| Skill | Описание |
|---|---|
| `start` | Стартира интерактивния помощник за избор на следваща стъпка |
| `init` | Инициализира нова папка за участие в поръчка |
| `eop-scan` | Сканира ЕОП портала за нови или актуални обществени поръчки |
| `eop-analyze` | Анализира документацията по поръчката и извлича ключови изисквания |
| `eop-price` | Генерира ценово предложение на база анализа |
| `eop-generate` | Генерира пълен комплект офертни документи |
| `eop-review` | Преглед и финална проверка на офертата преди подаване |

---

## Бърз старт

Типичен работен процес за участие в поръчка:

```
/eop-scan        # Намери подходящи поръчки
/eop-analyze     # Анализирай документацията
/eop-price       # Изгради ценово предложение
/eop-generate    # Генерирай офертните документи
/eop-review      # Провери и финализирай офертата
```
