# BloxPowers

Добре дошли в BloxPowers — вашият инструмент за автоматизирано участие в обществени поръчки.

## Предварителни изисквания

Преди да започнете, уверете се че имате инсталирани:
- Playwright MCP плъгин (за автоматизация на eop.bg)
- `python-docx` (`pip install python-docx`) — за работа с DOCX файлове
- `openpyxl` (`pip install openpyxl`) — за работа с Excel/КСС файлове
- LibreOffice (`libreoffice --headless`) — за конвертиране в PDF

## Първоначална настройка

Провери дали съществува `./bloxpowers/company-profile.md`.

Ако файлът **не съществува**:
> "Нямате фирмен профил. Стартирайте `/init` за първоначална настройка на вашата компания."

Ако файлът **съществува**, продължи с показване на менюто.

## Налични инструменти

### ЕОП — Обществени поръчки

| Команда | Описание |
|---------|----------|
| `/eop-scan` | Сканиране на eop.bg за отворени строителни поръчки |
| `/eop-analyze <offer-id>` | Анализ на конкретна поръчка — изтегляне и парсване на документация |
| `/eop-price <offer-id>` | Интерактивно ценообразуване — труд, материали, надценка |
| `/eop-generate <offer-id>` | Генериране на офертен документ (DOCX + КСС) |
| `/eop-review <offer-id>` | Финален преглед, корекции и PDF конвертиране |

### Настройки

| Команда | Описание |
|---------|----------|
| `/init` | Първоначална настройка на фирмен профил |

## Типичен работен процес

```
1. /eop-scan              → Намерете и изберете подходящи поръчки
2. /eop-analyze <id>      → Анализирайте изискванията и документацията
3. /eop-price <id>        → Определете цените за всяка позиция
4. /eop-generate <id>     → Генерирайте офертните документи
5. /eop-review <id>       → Прегледайте, коригирайте и финализирайте
```

## Работна директория

Всички данни се записват в `./bloxpowers/`:
- `company-profile.md` — фирмен профил
- `offers/<offer-id>/` — данни за всяка поръчка
