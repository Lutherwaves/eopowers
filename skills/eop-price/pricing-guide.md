# Методология за ценообразуване на строителни обществени поръчки

## Средни дневни ставки — регион Стара Загора (2024-2026)

| Квалификация | Дневна ставка (лв.) | Забележка |
|-------------|---------------------|-----------|
| Общ строителен работник | 80–120 | Неквалифициран труд |
| Квалифициран работник (зидар, мазач, арматурист) | 120–180 | С професионален опит |
| Специалист (електротехник, ВиК, заварчик) | 150–200 | Със сертификат |
| Инженер / технически ръководител | 200–300 | Висше образование, опит |

## Надценки за материали

- Стандартна надценка: 10–20% върху доставната цена
- При големи количества: 5–10% (обемна отстъпка)
- При складова наличност: 0% надценка (вече закупени)

## Общи разходи (overhead)

- Стандартен процент: 10–15% от директните разходи
- Включва: офис разходи, счетоводство, застраховки, амортизация

## ДДС

- Ставка: 20% (винаги се посочва отделно)
- В КСС: цените обикновено са БЕЗ ДДС
- В крайната оферта: посочвайте и двете стойности

## Стратегия за конкурентно ценообразуване

1. **Целете 70–85% от прогнозната стойност** — балансира между конкурентност и печалба
2. **Не слагайте под себестойност + 5%** — минимална печалба за покриване на рискове
3. **Ако критерият "цена" тежи >50%** — ценете по-агресивно (70–75%)
4. **Ако "техническо предложение" тежи повече** — инвестирайте в качеството на предложението, ценете умерено (80–85%)
5. **Проверете дали има минимална цена** — някои поръчки имат праг за "необичайно ниска цена" (~70% от прогнозната)

## Формула за единична цена

```
Единична цена = (Труд + Материали + Допълнителни) / Количество

Труд = Работници × Дни × Дневна ставка
Материали = Σ(Количество × Единична цена за материал)
Допълнителни = Транспорт + Механизация + Застраховки + Overhead
```

## Типични доставчици в Стара Загора

- **Bagira** — строителни материали, инструменти
- **HomeMax** — довършителни материали, бои, лепила
- **Praktis** — широк асортимент строителни материали
- **Praktiker** — инструменти, крепежни елементи
- **Локални доставчици** — бетон, арматура, тухли (търсете по конкретен материал)

## Бърз (batch) режим на ценообразуване

Подходящ за поръчки с 20+ позиции. Вместо позиция-по-позиция:

1. **Оценка на себестойност по раздели** — за всеки раздел, сумирай:
   - Труд: количество × часова норма × средна часова ставка
   - Материали: от пазарно проучване
   - Механизация: наеми + горива (ако е приложимо)

2. **Единен множител** — изчисли `множител = целева_сума / обща_себестойност`
   - Множителят покрива: overhead, непредвидени, печалба
   - Типични стойности: 1.15x–1.35x върху себестойност

3. **Единични цени** — за всяка позиция: `ед. цена = себестойност × множител / количество`

## Формула за анализ на единични цени (П1-П5)

Използва се за генериране на `price-analyses.xlsx`. Параметрите П1-П5 се зареждат от domain.md.

```
Единична цена = (Труд + Механизация + Материали) × (1 + П4) × (1 + П5)

Където:
  Труд = Часове × П1 × (1 + П2)
    П1 = средна часова ставка (лв./час)
    П2 = допълнителни трудови разходи (осигуровки, облекло и др.)

  Механизация = Машиночасове × Цена/час × (1 + П3)
    П3 = допълнителни механизация разходи (ремонт, поддръжка)

  Материали = Σ(Количество × Единична цена за материал)

  П4 = доставно-складови разходи (% от директните)
  П5 = печалба (% от себестойност + П4)
```

### Обратно изчисляване от крайна цена

Когато имаш крайна единична цена (от batch режим) и трябва да попълниш П1-П5 анализ:

1. Приеми разпределение: ~45% труд, ~10% механизация, ~45% материали (коригирай по позиция)
2. `себестойност = ед_цена / ((1 + П4) × (1 + П5))`
3. `труд_лв = себестойност × 0.45`
4. `часове = труд_лв / (П1 × (1 + П2))`
5. `механизация = себестойност × 0.10 / (1 + П3)`
6. `материали = себестойност × 0.45`
